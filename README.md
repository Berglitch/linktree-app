# TI Rede Amazônica - Linktree

Painel unificado de acesso rápido aos sistemas oficiais, planilhas e formulários de uso diário pela equipe de TI da Rede Amazônica.

## 🚀 Como Hospedar e Gerar um Link para Compartilhar

Como este projeto já é um repositório Git público no seu GitHub (`Berglitch/linktree-app`), a forma mais fácil, rápida e gratuita de colocá-lo online é usando o **GitHub Pages**.

Siga o passo a passo abaixo para ativar:

1. Acesse o seu repositório no GitHub: [https://github.com/Berglitch/linktree-app](https://github.com/Berglitch/linktree-app)
2. No menu superior, clique na aba **Settings** ⚙️ (Configurações).
3. No menu lateral esquerdo, clique em **Pages** (Páginas).
4. Sob a seção **Build and deployment**:
   - No campo **Source**, certifique-se de que está selecionado **Deploy from a branch**.
   - No campo **Branch**, clique no menu de seleção (que pode estar como *None*) e escolha **main**.
   - Mantenha a pasta como **/(root)**.
   - Clique no botão **Save** (Salvar).
5. Aguarde cerca de 1 a 2 minutos. 
6. Atualize a página. No topo da seção **Pages**, aparecerá uma mensagem de sucesso com o link do seu site publicado!
   - O link será algo como: `https://berglitch.github.io/linktree-app/`

Agora você pode copiar esse link e enviar para seus amigos e colegas de trabalho!

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da aplicação.
- **CSS3**: Estilização responsiva com efeito premium (Glassmorphism, gradientes e animações de hover).
- **JavaScript (Vanilla)**: Lógica de filtros de busca dinâmicos, alternância de tema escuro/claro e funcionalidade de compartilhamento.
